Learn HTML

Introduction


The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript. HTML Is used in developing of webpages and websites. Web browsers receive HTML documents from a web server or from local storage and render the documents into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document. It was Developed by Tim Berners Lee. HTML 5 is the Latest Version.

HTML uses Tags and attributes. Tags are enclosed by a pair of angle brackets. There are 2 types of Tags - closed tag (<p> </p>) and opened tags (<img>). Closed tags requires a closing tag in a pair of angle brackets with a slash (</tagname>). Each Attribute has a parameter which is written in "" after equal to (=) symbol. <tag attribute = "parameter">. There are 2 types of attributes - Global and Event Attributes.

In this lesson, We will learn the basics of HTML and CSS. You can Start creating your own webpages! So why wait??? Let us get started. (Use the side pane to navigate to the desired topic)



Basic HTML Tags


Tags Use
<!DOCTYPE html> Every webpage starts with the DOCTYPE declaration.
<html> </html> The starting and the ending tagof the html document
<head> </head> Anything written in the head tag. The style, title and link tags are written within the head tag.
<title> </title> Used to give title to the webpage. Located within the head tag
<body> </body> The main tag of the html document. The content written in the body tag will be seen. Written after the head tag.
<!--Comment --> The content written inside <!-- --> will not be seen


Write HTML using notepad and VS code


To Write a html Document, you can use any code editing apps. The simplest way of writing a html document is using notepad. You can also use other apps like VS Code, etc. To begin writing, open a new notepad file. Type the codes. Save it with .html extension and later view the file on the browser.

Visual Studio code can also be used instead of notepad. VS code is a much easier way than notepad as it helps to avoid errors and automatically closes tags. The file should be saved with .html extension. Both html and CSS codes can be written in notepad as well as vs code



H1...H6 and p tags


<h1> </h1>, <h2> </h2>, <h3> </h3>, <h4> </h4>, <h5> </h5>, <h6> </h6> are the heading tags. H1 is the largest and h6 is the smallest. The heading will automatically appear bold.
The <p> </p> is used to type paragraphs. The font in p tag is not bold and appears smaller than the headings



BR, HR and PRE


BR, HR and PRE are used to give line breaks. <br> and <hr> are opened tags while
<pre> </pre> is a closed tag. BR is used to give line break while hr is used to draw a horizontal line break. Text in a <pre> </pre> element is displayed in a fixed-width font, and the text preserves both spaces and line breaks. The text will be displayed exactly as written in the HTML source code.



Superscript and Subscript


The <sup> </sup> tag is used to write the letters in super script. The content inside the sup tag will be raised above the normal font. <sub> </sub> is used to give subscript to the content inside the tags. The content will be little below the original font.

Bold , Italics , Underline and Scratch


The <b> </b> tag is used to write bold text. The other tag for bold text is the
<strong> </strong>. The <i> </i> tag is used to type slanted letters. The other tag used is the
<em> </em> tag. The <u> </u> tag is used to underline the content. The <s> </s> tag is used to scratch the text



Attributes


HTML attributes provide additional information about HTML tags. All HTML tags can have attributes. <tagname attributename="value"> </tagname>. The value of the attribute should always be in

There are 2 types of attributes - global and event Attributes. The global attributes are attributes that can be used with all HTML elements. HTML has the ability to let events trigger actions in a browser, like starting a JavaScript when a user clicks on an element. The attributes used for this are called Event Attributes

Some Global Attributes are -

Style attribute - This is used to write CSS for an HTML Element.
Class attribute - It is used to specify a class for an HTML element. Multiple elements can have the same class name.
ID attribute - It is used to specify a unique id for an HTML element. You cannot have more than one element with the same id in an HTML document.

Lists


HTML lists allow web developers to group a set of related items in lists. There are three types of lists - ordered list, unordered list and description lists. A list can be written within another list. This is called nested list.

Tags Tags used within the main tag Attributes Characterisics and Use
<ol> </ol> <li> </li> defines a list item. tag is used inside ordered lists(ol), unordered lists (ul) The type attribute can be used to define the type of list.
  • type = "1" - Displays numbers (default)
  • type = "I" - Displays Capital Roman numbers
  • type = "A" - Displays Capital Letters.
  • type = "a" - Displays Lowercase alphabets
  • type = "i" - Displays Lowercase Roman numbers
 The HTML ol tag defines an ordered list. An ordered list can be numerical or alphabetical. The list items should be placed in a particular order
<ul> </ul> <li> </li> defines a list item. tag is used inside ordered lists (ol), unordered lists (ul) The type attribute can be used to define the type of list.
  • type = "disc" - Item Marker ●
  • type = "circle" - Item Marker ○
  • type = "square" - Item Marker ■
  • type = "none" - No Item Marker
 The <ul> tag defines an unoredered or bulleted list. The list items can be placed in any order.
<dl> </dl> <dt> </dt> defines the term name. <dd> </dd> defines the definition of the term - The <dl> tag defines a definition or description list.


Images


The <img> tag is used to insert an image. It is an opened tag so it does not reuire a closing tag. Ther attributes used in the <img> tag are -

Attribute Use Value
src Used to specify the source of the image URL of the image
height Used to define the height of the image Number (in pixels)
width Used to define the width of the image Number (in pixels)
alt Used to give an alternative text if the browser fails to load the image text
title Used to give a title to the image. The title is seen when the cursor is hovered over the image text

Bookmarks


Bookmarks allow readers to jump to specific parts of a web page. Bookmarks can be useful if a web page is very long. To create a bookmark - first create the bookmark, then add a link to it. When the link is clicked, the page will scroll down or up to the location with the bookmark.

First, use the id attribute in the desired tag to create a bookmark. Create a hyperlink with the value of href as id name. A '#' Should be written before the id name in the hyperlink.

Example - <a href = "#id_name"> HYPERLINK </a>. When the Hyperlink is clicked, it will take to the place with the id "id_name"

Div and Span


The <div> </div> tag defines a division or a section in an HTML document. The div tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript. The div tag is easily styled by using the class or id attribute. It also supports other attributes.

The <span> </span> tag is an inline container used to mark up a part of a text, or a part of a document. The span tag is much like the div element, but div is a block-level element and span is an inline element. The tag is easily styled by CSS or manipulated with JavaScript using the class or id attribute.



Link CSS in HTML


There are three ways to insert CSS in HTML -

  • Internal - Use the <style> </style> tag inside the head tag
  • Inline - Use style attribute inside the element
  • Extrenal - Use <link> tag inside the head tag. Syntax: <link rel = "stylesheet" href = "NameOfFile.css">. One has to create a new file for external css
